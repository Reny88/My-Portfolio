Project X Ball Game

This project is an interactive X Ball game built with HTML5 Canvas, CSS, and JavaScript. The game features a bouncing ball and a paddle controlled by the player. The objective is to break all the bricks without letting the ball fall below the paddle. This project helped me understand game mechanics and the use of Canvas for creating dynamic graphics.
